## LatteIsAHorse_Backend
퀴즈를 풀며 캠퍼스 생활에 관련된 정보와 리워드(라떼 스택)를 얻을 수 있는 서비스입니다.      
     

### Develop Settings   
- Java 8   
- SpringBoot 2.5.6   
- H2 database 1.4.200   
- Querydsl 4.4.0   
- Redis 6.0.9   
- Firebase 7.0.1

### Usage     
```sh
$ ./gradlew clean build
```
   
Application을 실행하기 전에 로컬 Redis를 실행시키세요.
